# Daily Challenge - Madza
Inspired by: https://dev.to/madarsbiss/coding-series-1-by-madza-26oi

## Description
Multiply a given number `x` by the sum of it's digits. Then subtract the reverse of `x`.

`x` will always be:
* `x > 0`
* `x % 10 != 0`
* `x % 1 == 0`

## Test Cases
calculate(1)
calculate(103)
calculate(-42)
calculate(0)
